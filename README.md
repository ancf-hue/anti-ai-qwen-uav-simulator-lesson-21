📋 Описание
Оператор Гонщик — это браузерный 3D-тренажёр управления БПЛА, в котором игрок соревнуется с AI-дроном на круговой трассе. 

Проект создан в рамках учебного курса Qwen (Урок 21) и демонстрирует возможности веб-технологий в создании интерактивных 3D-симуляторов.

⚠️ Главное правило: Столкновение с деревом = конец гонки! Пролетите через все ворота быстрее AI-дрона!

✨ Возможности

🎮 Геймплей

🏎️ Гонка против AI — умный бот, летящий по оптимальной траектории

🚩 Система ворот — от 5 до 12 контрольных точек в зависимости от сложности

🌳 Динамические препятствия — деревья, расположенные по всей трассе

⏱️ Таймер и рекорды — сохранение лучшего времени в localStorage

🎯 3 уровня сложности — Лёгкий, Средний, Сложный

🎨 Визуал и звук

🌅 Кинематографичное освещение — ACES Filmic tone mapping, мягкие тени

🚁 Детализированная модель дрона — карбоновый корпус, моторы, пропеллеры, камера

🎵 Процедурные звуковые эффекты — Web Audio API (ворота, авария, победа)

🌫️ Атмосферный туман — FogExp2 для глубины сцены

💡 Неоновая подсветка — динамические PointLight на дронах

🖥️ Интерфейс

📊 HUD с телеметрией — высота, скорость, сигнал, дистанция до AI

🔔 Система уведомлений — всплывающие сообщения о событиях

📱 Адаптивный дизайн — корректная работа на мобильных устройствах

🎭 Модальные окна — победа/поражение с детальной статистикой

🛠️ Технологии

Основные

HTML5 — семантическая разметка, доступность (a11y)

CSS3 — градиенты, анимации, backdrop-filter, @keyframes

JavaScript (ES6+) — модульная архитектура, классы, async/await

Three.js r128 — 3D-рендеринг, физика, освещение

Инструменты

Web Audio API — генерация звуков в реальном времени

localStorage — сохранение рекордов

requestAnimationFrame — плавная анимация 60 FPS

Pointer Lock API — управление камерой мышью
